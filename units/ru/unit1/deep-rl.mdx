# “Глубина” в обучении с подкреплением [[deep-rl]]

<Tip>
То, о чем мы говорили до сих пор, - это обучение с подкреплением. Но где же здесь "Глубина (Deep)"?
</Tip>

Глубокое обучение с подкреплением (Deep Reinforcement Learning) использует **глубокие нейронные сети для решения задач обучения с подкреплением** - отсюда и название "глубокое".

Например, в следующем блоке мы познакомимся с двумя алгоритмами, основанными на ценности: Q-Learning (классическое обучение с подкреплением) и затем Deep Q-Learning.

Разница заключается в том, что в первом случае **мы используем традиционный алгоритм** для создания Q-таблицы, которая помогает нам определить, какое действие следует предпринять для каждого состояния.

Во втором подходе **мы будем использовать нейронную сеть** (для аппроксимации значения Q).
